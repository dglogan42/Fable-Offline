## Why freight belongs in urban planning

Cities move **people and goods**. Plans that only optimise passenger modes can:
- Undermine port/industrial access  
- Push heavy vehicles onto inappropriate local streets  
- Worsen safety and equity outcomes on freight corridors  
- Miss mode-shift opportunities (rail, coastal)  

Freight planning links **land use** (where generators sit) to **networks** (which links are strategic) and **operations** (when and how trucks move).

## Building blocks

| Block | Planner questions |
|-------|-------------------|
| Demand & generators | Port, logistics parks, industrial, construction, retail DC |
| Network hierarchy | Strategic freight vs supporting |
| Mode | Road HGV, rail, coastal, air |
| Land use | Industrial zoning vs sensitive receivers |
| First/last mile | Loading, HPMV, height/weight limits, town-centre access |
| Time horizon | Current vs first decade / long-term |
| Problems | Safety, delay, emissions, amenity, equity |
| Opportunities | Consolidation, mode shift, protected corridors |
| Intermodal | Port-rail, rail-road terminals |
| Evidence | Counts, models, crashes, complaints — dated sources |

---

## Evaluation criteria (default)

1. Economic access / reliability  
2. Safety  
3. Amenity & equity  
4. Environment  
5. Network resilience  
6. Land-use compatibility  
7. Deliverability
