# Outfit selector & create — framework

**Skill:** `outfit-selector-create`  
Works with wardrobe picks **and** sew-from-scratch via Seamly.

---

## Select rubric (closet)

| Axis | Prefer |
|------|--------|
| Occasion match | Dress code / activity |
| Weather | Layer reality |
| Colour story | 1–2 accents max for “styled” |
| Silhouette | Intentional shape |
| Comfort / confidence | User’s stated priority wins |
| Photo potential | Clean lines for feed if posting |

---

## Create brief fields

- Name, occasion, vibe  
- Make vs buy vs thrift per piece  
- Pattern path: Seamly original / licensed commercial / mashup of **user-owned** blocks  
- Skill level honesty (first zipper vs advanced)  
- Time budget  

---

## Measurement ethics

- User measures themselves or a trusted person with consent  
- No ranking bodies  
- Client measurements → `_local/` only  

---

## Hand-off to Instagram

When the outfit is worn or on a dress form:

1. `instagram-selfie-selector` → **fit-check** / **select-hero**  
2. Caption can mention #memade if handmade (user choice)  
3. No auto-post
