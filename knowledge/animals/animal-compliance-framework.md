# Animal compliance agent — framework

**Skill:** `animal-compliance-agent`  
**Not legal, veterinary, or animal-control advice.**

---

## Owner checklist (dogs)

- Registration current  
- Microchip  
- Control in public  
- Secure property  
- Waste  
- Know local rules  

---

## Routing

| Issue | Channel |
|-------|---------|
| Attack / danger now | **111** |
| Roaming / registration / bylaws | Local council animal services |
| Welfare (national primary) | MPI channels as published |
| Export of animals/products | MPI Exporter Help |
| Rental pet dispute | Tenancy Services + property-manager-agent |
