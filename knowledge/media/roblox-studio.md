## Product summary

**Roblox Studio** is a free Windows/Mac application to create immersive 3D games/experiences for the Roblox platform (multi-device players). Includes building, scripting (Luau), testing, and publishing tools. Studio ships with the same engine family as the player app; **keep Studio updated** because experiences run on the latest engine.

| Spec | Minimum | Recommended |

| Resolution | — | 1600×900+ |

**Creation devices (education FAQ):** PC or Mac with internet. **Studio cannot run on Chromebooks or phones** for development (play client may differ).

## Installation (setup doc)

1. Download Studio from official setup/docs flow  
2. Confirm **Download Studio** in dialog  
3. Run `RobloxStudio.exe` (Windows) or `RobloxStudio.dmg` (Mac)  
4. **Launch Studio**  
5. Sign in with Roblox account (create at roblox.com if needed)

## Customization (setup)

- Toolbar custom tabs  
- Window layout  
- File → Customize Shortcuts  
- Studio Settings: Alt+S (Win) / ⌥S (Mac); search e.g. theme

## Updates & betas

- **Update** button when Studio is outdated — save/close place, update, restart  
- **File → Beta Features** — enable, Save, restart
