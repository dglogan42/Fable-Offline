## What the desktop app does

Per Adobe marketing/help themes:

- Download, install, update Creative Cloud **desktop** apps  
- Launch apps from one place  
- Sync/share files and **Libraries**  
- Fonts, Adobe Stock (subscription-dependent)  
- Behance showcase (optional)  

Web apps (e.g. some Express/Firefly surfaces) may not need desktop install.

---

## Install more apps

1. Open **Creative Cloud desktop**  
2. **Apps** tab  
3. Find app (Photoshop, Lightroom, …) → **Install**  
4. Sign-in with licensed Adobe ID  

Previous versions: More actions → Other versions (Help: install previous versions) when a project requires pin.

Install location: Account → Preferences → Apps → Install location (Help: change install location).

## Fable pipeline role

| App | Build role |
|-----|------------|
| Lightroom / Lightroom Classic | Cull + develop + batch export |
| Photoshop | Actions, composites, thumbnails |
| Premiere / After Effects | Optional motion (if licensed); else CapCut/Resolve |
| Media Encoder | Batch encodes when using Adobe video apps |

Pair with CapCut / DaVinci Resolve for non-Adobe edit paths — see `creative-pipeline-builds.md`.
