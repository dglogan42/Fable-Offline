## Product summary (official + wiki)

**Minecraft Education** (formerly **Minecraft: Education Edition**) is an educational version of **Bedrock Edition** for classroom use (wiki). Developed by **Mojang Studios** and **Xbox Game Studios**.

- Announced 2016-01-19; full release **2016-11-01** (Windows & macOS)  
- Platform rollouts (wiki seeds): iPad 2018-09; iPhone 2022-08; ChromeOS 2020-08; Android 2022-08  
- **Not** the discontinued Java mod **MinecraftEdu** (TeacherGaming)  

Official hub adds: lesson library, standards-aligned curricula, Classroom Mode companion app, Code Builder, Immersive Reader marketing, Microsoft 365 education licensing.

## Exclusive / classroom features (wiki)

Main article: [Minecraft Education exclusive features](https://minecraft.wiki/w/Minecraft_Education_exclusive_features).

| Feature | Classroom use |
|---------|----------------|
| **Classroom collaboration** | Whole-class multiplayer (wiki: no hard student-count limit on one server — still subject to network/device limits) |
| **NPC** | Guides with instructions; can open web links |
| **Camera + Portfolio** | Screenshots and project documentation |
| **Chalkboards** | In-world goals/instructions (sizes 1×1, 2×1, 3×2) |
| **Tutorial world** | Navigation, crafting, place/break basics |
| **Allow / Deny blocks** | Zones where students can or cannot build |
| **Border blocks** | Restrict movement (X/Z influence; wiki notes some bypasses e.g. ender pearls) |
| **Classroom Mode** | Companion app: map overview, player locations, chat monitor, some world toggles |
| **Code Builder + Agent** | Students write code; Agent executes in-world |
| **`/ability`** | Per-player permissions (e.g. World Builder, MayFly, mute) |
| **Immutable World gamerule** | Non-builders cannot place/break except on Allow blocks |
| **Chemistry set items** | Education exclusives (e.g. ice bombs, underwater TNT/torches via Chemistry Set) |

World create UI also includes **Classroom**, **Cloud Storage**, **Multiplayer**, **Coding & Scripting** tabs (wiki Ore UI gallery) — UI labels VERIFY LIVE.

## Education vs retail / Java

| | Minecraft Education | Bedrock / Java retail |
|--|---------------------|------------------------|
| Codebase | Bedrock-based | Bedrock or Java |
| Classroom tools | Yes (NPCs, borders, Classroom Mode, Agent…) | No school suite |
| macOS | Official Bedrock-family Education on macOS (wiki trivia: only official Bedrock-derived macOS release) | Java on macOS; Bedrock availability differs |
| License | Education / M365 education | Retail / Game Pass / etc. |

Predecessor: **MinecraftEdu** (discontinued Java mod) — not the same product.
