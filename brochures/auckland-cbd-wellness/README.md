# Auckland CBD Wellness Retreat brochure

Curated **self-care / wellness-retreat** program for Auckland city centre, based on the public [Heart of the City Health & Wellbeing](https://heartofthecity.co.nz/health-wellbeing) directory.

| File | Role |
|------|------|
| `Auckland_CBD_Wellness_Retreat_Brochure.pptx` | 8-slide brochure |
| `build_brochure.js` | Regenerate with pptxgenjs |
| Skill | `hotc-wellness-retreat-kit` |
| Knowledge | `knowledge/health/auckland-cbd-wellness-retreat.md` |

## Rebuild

```bash
npm.cmd install pptxgenjs
node brochures/auckland-cbd-wellness/build_brochure.js
```

**Not medical advice.** Not an official Heart of the City package.
