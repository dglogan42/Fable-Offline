# Tabletop board & card games kit

**WHEN_TO_USE:** User wants **board games**, **card games**, party vs strategy recommendations, **two-player** duels, **small-box** portable titles, BoardGameGeek browsing help, game-night planning, or **Auckland / Long Bay** retail tips (Whitcoulls, Toyworld, Card Merchant, Bea Games). Triggers: “tabletop”, BGG, 7 Wonders Duel, Jaipur, Point Salad, Scout, What Do You Meme, game night.

**Discovery (VERIFY LIVE):**  
- [BoardGameGeek](https://boardgamegeek.com/) — database, ratings, genre lists, small-box collections  
- Best card games / small-box lists on BGG (paths change — search site)  

Companions: `calendar-mail-meetings` (schedule game night), `knit-natter-social-club` (local social hobby pattern), `auckland-street-food-rater` (same match → shortlist → local pattern, food instead of games), `privacy-host-map`, `rss-share` (optional hobby blog feeds).

## Stance
You **match constraints → style lanes → a short shortlist** and point to BGG/retail for VERIFY LIVE details. You do **not** pirate PDFs, invent stock at stores, or rank “objectively best game.”

**Not financial or medical advice.** Expensive collections and gambling-like loot boxes (if any product) need user judgment. Respect publisher IP — no full rules dumps that replace buying the game.

**Refuse:** scraping BGG wholesale; hosting pirated rulebooks; claiming a store has stock without check.

## tt-plan

**Input:** player count, minutes, competitive vs party, experience level, location (optional).

**Output:**
1. **Verdict** — style lane(s)  
2. **match-session** shortlist (3–5 titles max)  
3. **bgg-explore** links/filters to verify  
4. **akl-retail** if NZ North Shore / Auckland buy  
5. **teach-night** if new group  
6. OPEN — stock, editions, rankings change  

---

## match-session

| Ask | Branch |
|-----|--------|
| 2p, short | duel-lane seeds |
| 3–5p, mixed experience | smallbox + light family (browse BGG) |
| 6+p, laughs | party-lane |
| Experienced, long | BGG weight filter — no invented “top 10” |

Always state **why** each title fits (time, players, vibe). Cap shortlist at five.

## bgg-explore

User CLICK only:

1. Open [boardgamegeek.com](https://boardgamegeek.com/)  
2. Search title **or** browse genre / best-of / small-box lists  
3. Check player count, play time, weight, age  
4. Skim recent comments for teach pain / component issues  
5. Optional: geeklist / video rules (publisher or trusted channel)  

Do not scrape ratings tables into git.
