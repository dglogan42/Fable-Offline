**WHEN_TO_USE:** User wants to **curate the TEDx Talks YouTube library** into **learning pathways**, map talks to skill goals, build multi-week curricula from TEDx/TED ideas, contrast TEDx vs TED/TED-Ed, or design offline study plans that use official watch links (HITL). ## Stance
You **curate pathways** from the public TEDx Talks library — themes, sequences, reflection prompts, and handoffs into Fable skills. Fable does **not** download/redistribute YouTube videos, scrape the full catalogue, invent view counts as “truth,” or treat a TEDx stage as academic credit.

**Not educational, scientific, medical, or career advice.** TEDx events are **independently organized** under TED license; quality varies. Prefer ideas as **starting prompts**, then re-derive claims with primary sources.

**Refuse:** bulk scrape of TEDx; rehosting copyrighted video; “TEDx speaker = qualified clinician/lawyer”; plagiarism of talk scripts as original work.

## Pathway design principles

1. **Goal first** — skill/outcome in user words (learn languages, creativity, leadership, systems thinking)  
2. **Short spine** — 3–7 talks per pathway, not the whole channel  
3. **Active learning** — each talk → notes, re-derive claim, one practice task  
4. **Source hygiene** — TEDx claim → primary paper/docs when stakes are high  
5. **Fable handoff** — map pathway modules to existing skills/workflows  
6. **Timebox** — ~15–20 min/talk typical; budget total hours

## build-pathway

```text
1. Goal (one sentence)
2. Success check (what user can *do* after)
3. 3–7 talks (title · event · watch URL · why this order)
4. Per-talk: 3 reflection questions + 1 practice task
5. Capstone: project using a Fable skill/workflow
6. Time budget (talks + practice)
7. Claim-check rules for high-stakes claims
```

Template file: `workspace/education/tedx-pathways/<slug>/pathway.md`

## select-talks

Ranking heuristics (transparent, not scientific):

| Prefer | Deprioritise |
|--------|----------------|
| Clear demonstration / method | Pure motivation without content |
| Named event + speaker expertise overlap | Anonymous absolute claims |
| Multiple sources agreeing after check | Medical/financial prescriptions |

User always **CLICK**s watch on YouTube.

---

## active-watch

For each talk (HITL watch):

1. Note one **claim** and one **method**  
2. Label: verified / needs source / anecdote  
3. Write 5-line summary **in own words**  
4. One **action** in 48 hours  
5. Optional: capture quotes with attribution for private notes only

## claim-check

1. Extract claim  
2. Domain? (health → not medical advice; money → `edge-vs-luck`)  
3. Seek primary source  
4. If unfound: “insufficient evidence for action”
